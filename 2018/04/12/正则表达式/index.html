正则表达式

发表于 2018-04-12 | 分类于 js笔记
字数统计: | 阅读时长 ≈

关于正则表达式，工作中用到的其实不多，常见的就是邮箱，手机号验证，总是用到了现搜，刚好今天完成了市场需求，等待另一个需求确认ing…有时间系统总结一下琐碎的正则表达式～慢慢补充

定义正则

正则常用方法

  • test()：在字符串中查找符合正则的内容，若找到返回true，反之返回false
    用法：正则.test(字符串)
    例如：判断是否是数字

正则表达式中有很多符号，代表不同的含义，利用它们我们可以定义不同的规则
|    符号    |    含义   | 
| :-------: | :------: |
| \s | 空格  |
| \S | 非空格 |
| \d | 数字 |
| \D | 非数字 |
| \w | 字符（字母，数字，下划线_） |
| \W | 非字符 |
| 开发阶段 | 开发时长 |

  • search()：在字符串搜索符合正则的内容，搜索到就返回出现的位置（从0开始，如果匹配的不只是一个字母，那只会返回第一个字母的位置），如果搜索失败就返回-1
    用法：字符串.search(正则)
    在字符串中查找符合正则的内容，忽略大小写：i–ignore(正则默认是区分大小写的，如果不区分大小写的话，在正则的最后加标识i)
    例如：在字符串中找字母b，且不区分大小写
